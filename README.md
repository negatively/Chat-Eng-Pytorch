## Built Using
- Python 3.8.8
- PyTorch
- NLTK
- Flask
- Javascript

## Usage
### Training
Run
```console
python train.py
```
This will dump `chat_data.pth` file. And then run

### Run Chatbot App
To run chatbot
```console
python app.py
```
